Here's the course structure with 9 Modules:

| Module | Lessons | Key Concepts |
|---|---|---|
| 1. What is Linux? | L01–L03 | History, distributions, kernel vs OS, open source |
| 2. The Terminal | L04–L07 | Shell, bash, command syntax, man pages |
| 3. Files & Directories | L08–L11 | Filesystem hierarchy, pwd/ls/cd/mkdir/rm, permissions overview |
| 4. Working with Files | L12–L14 | cat/less/head/tail, nano, cp/mv, wildcards |
| 5. Users & Permissions | L15–L17 | Users and groups, chmod/chown, sudo, passwd |
| 6. Processes | L18–L20 | ps/top/kill, jobs, &, systemd basics |
| 7. Networking Basics | L21–L23 | ping/curl/wget, ssh, IP basics, /etc/hosts |
| 8. Package Management | L24–L25 | apt/apt-get, dpkg, updating system |
| 9. Shell Scripting** | **L26–L29 | Shebang & script structure, variables & conditionals, loops, functions & task automation|
